Project: Week 0

Objectives

  • Become more comfortable with HTML and CSS to design and style webpages.
  • Learn to use SCSS to write more complex stylesheets for your webpages.

    • Description

    This project was made to become the basis of a book promotion project. On the initial page, I put a table to enumerate the books and their respective authors that would be addressed on the page. Below I applied the summaries of each book as a list and added static images for each one.
    The user has the option to read more content about the books by clicking on "See More".
    When entering each review page, the other books will be listed for the user to check other reviews if he wants.
    In the future, this website template may be more elaborate, using comments, user registration, and notes system.
